# ToDoApp - Webフロントエンド

React + TypeScript + Viteで構築されたToDoアプリケーションのWebフロントエンドです。

## 機能

### 実装済み機能
- ✅ タスクの作成・編集・削除
- ✅ タスクステータス管理（未着手・進行中・完了）
- ✅ 優先度設定（1-5）
- ✅ 進捗管理（0-100%）
- ✅ フィルタリング（すべて・未着手・進行中・完了・期限切れ・近日期限）
- ✅ 進捗統計ダッシュボード
- ✅ レスポンシブデザイン

### API統合
- すべてのバックエンドAPIエンドポイントと統合
- カテゴリ管理API
- タグ管理API
- リマインダーAPI
- 進捗管理API

## セットアップ

### 前提条件
- Node.js 18以上
- npm または yarn
- バックエンドサーバーが起動していること（http://localhost:8000）

### インストール

```bash
# 依存関係のインストール
npm install
```

### 開発サーバーの起動

```bash
# 開発モードで起動
npm run dev
```

アプリケーションは http://localhost:3000 で起動します。

### ビルド

```bash
# 本番用ビルド
npm run build

# ビルドのプレビュー
npm run preview
```

## 使い方

### タスクの作成
1. 「+ 新規タスク」ボタンをクリック
2. タイトル、説明、優先度、ステータス、進捗を入力
3. 「作成」ボタンをクリック

### タスクの編集
1. タスクカードの「編集」ボタンをクリック
2. 必要な情報を変更
3. 「更新」ボタンをクリック

### タスクのステータス変更
- 「開始する」ボタン: 未着手 → 進行中
- 「完了にする」ボタン: 進行中 → 完了
- 「未完了に戻す」ボタン: 完了 → 未着手

### フィルタリング
サイドバーのビューボタンをクリックして、タスクをフィルタリングできます：
- すべてのタスク
- 未着手
- 進行中
- 完了
- 期限切れ
- 近日期限（7日以内）

## APIプロキシ設定

開発環境では、Viteのプロキシ機能を使用してバックエンドAPIにアクセスします：

```typescript
// vite.config.ts
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8000',
      changeOrigin: true,
    }
  }
}
```

## トラブルシューティング

### バックエンドに接続できない
- バックエンドサーバーが起動していることを確認してください
- バックエンドが http://localhost:8000 で起動していることを確認してください

### ビルドエラー
```bash
# node_modulesを削除して再インストール
rm -rf node_modules
npm install
```
